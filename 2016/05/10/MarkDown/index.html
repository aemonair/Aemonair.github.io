This is the Package Control about install the Markdown at the Subline Text 3

[TOC]

Sublime Text 是一个轻量、简洁、高效、跨平台的编辑器。具有良好的扩展功能，官方称之为安装包(Package)。

在各种环境下，使用Sublime Text编辑Markdown也是一个很好的选择。
由于sublime text提供了对Markdown语言的支持。通过设置可实现markdown预览和转换功能。

一、下载安装Sublime Text

Sublime Text：The text editor you’ll fall in love with

二、安装Package Control
I.package control安装
  1. 按 Ctrl + ` 打开console，(Or 从菜单View - Show Console)，出现控制台。
  2. 粘贴以下 Python 代码到console并回车运行，不出意外即完成安装。当看到代码最后一行提示的时候说明安装成功。
  3. 重启Sublime Text ，若安装成功可在Preferences->Package Settings看到Package Control。

II.手动安装

  1. 从这里Package control or (https://packagecontrol.io/Package%20Control.sublime-package)下载到package control的包;

  2. 然后将其放到sublime text 的安装包存放位置的一个叫Installed Packages的目录下，（点击 Preferences> Browse Packages）;

  3. 对了，然后重启，就可以在Preferences菜单下看到package settings和package control两个子菜单了

  • 这里要注意的一点是：Installed Packages目录可以通过Preferences菜单下的Browse Packages的子菜单找到，点击这个子菜单会弹出一个目录，这个是sublime text存放安装的插件的地方，但是Package control不是这里的，这个目录的上一个目录有Installed Packages目录，这里才是存放Package control的正确地方
  • 安装好Package control我们就可以轻松的安装其它插件了，但还是有个代理的问题，这个可以通过Preferences -> Package Settings -> Package control -> Settings - User配置文件修改解决，在配置文件中添加两项：
    "http_proxy": "http://web-proxy.example.com:8080", "https_proxy": "http://web-proxy.example.com:8080",
  • 详情据官网介绍：Installation -Package Control
三、安装 Markdown Preview
  1. 按Ctrl + Shift + P，调出命令面板。
  2. 输入pcin 后找到 (Package Control: Install Package)回车。
  3. 搜索markdown preview，点击安装。
四、使用Sublime Text + Markdown preview

Markdown Preview较常用的功能是preview in browser和Export HTML in Sublime Text，前者可以在浏览器看到预览效果，后者可将markdown保存为html文件。

I.Markdown语法编辑文档
  1. Ctrl + n 新建一个文档, Ctrl + o 打开一个文档。
  2. 按Ctrl + Shift + P，输入ssm 后回车(Set Syntax: Markdown)，使用Markdown语法编辑文档。
II.在浏览器预览Markdown文档
  1. 按Ctrl + Shift + P
  2. 输入mp 后回车(Markdown Preview: current file in browser)
  3. 此时就可以在浏览器里看到刚才编辑的文档了
    preview in browser是实时的，需要在st保存，然后浏览器刷新才能看到新的效果.
III、快捷键

sublime text支持自定义快捷键，markdown preview默认没有快捷键，我们可以自己为preview in browser设置快捷键。
方法是在Preferences ->Key Bindings - User打开的文件的中括号中添加以下代码(可在Key Bindings Default找到格式)：

{ "keys": ["alt+M"], "command": "markdown_preview", "args": { "target": "browser"} }

"alt+M"可设置为自己喜欢的按键。

IV、设置语法高亮和mathjax支持
  1. 在Preferences ->Package Settings->Markdown Preview->Setting - Default中的第47行和57行找到
  1. 将 两个false改为true即可。语法高亮跟编辑器的主题有关，可以在Preferences ->Color Scheme找自己喜欢的主题。
